🚀 Cyber-Secure Insurance Fraud Detection & Premium Prediction using ML
📌 Project Overview

This project focuses on building a machine learning-based system to detect fraudulent insurance claims and predict monthly insurance premiums. It integrates cybersecurity concepts to ensure secure data handling and reliable system performance.

Insurance companies face major challenges due to:

Increasing fraudulent claims 💸 Difficulty in accurately predicting customer premiums 📊

This system aims to solve both problems using ML models + secure architecture.

🎯 Objectives Detect fraudulent insurance claims using classification models Predict monthly premiums using regression models Ensure secure data processing using cybersecurity practices Improve accuracy and efficiency in insurance analytics 🧠 Technologies Used Python 🐍 Machine Learning (Scikit-learn) Pandas, NumPy Matplotlib / Seaborn Cybersecurity Concepts (Authentication, Data Validation) 📂 Dataset Information Dataset Name: Insurance Fraud Detection (by arpan129) Features: 39 attributes Target Variable: fraud_reported (Y/N) Feature Categories: Customer Demographics (age, education, occupation) Policy Details (premium, tenure, deductibles) Incident Details (type, severity, location) Financial Data (claim amounts) ⚙️ Data Preprocessing Steps Handling missing values Data cleaning & normalization Feature engineering (date conversion) Encoding categorical variables Dropping irrelevant features Final dataset: Preprocessed_Fraud_Detection_Data.csv 🤖 Machine Learning Models 🔍 Fraud Detection (Classification) Decision Tree Random Forest

Accuracy:

Random Forest: ~80% – 90% ✅ Decision Tree: ~70% – 85% 💰 Premium Prediction (Regression) Linear Regression Random Forest Regressor
